== Overview

FusionAuth supports WebAuthn and passkeys, a new way to securely authenticate users without requiring passwords.

image::passwordless/webauthn/reauth-enable.png[Example of WebAuthn prompt screen,width=1200,role=bottom-cropped]

At this time, FusionAuth supports first factor authentication with WebAuthn. It can be configured on a per-tenant basis.

Users may also associate WebAuthn passkeys to increase the security of their account using account management functionality.

image::account-management/bootstrap-prompt.png[Example of WebAuthn passkey screen,width=1200,role=bottom-cropped]

Learn more about link:/docs/v1/tech/admin-guide/webauthn[configuring WebAuthn].

Learn more about link:/docs/v1/tech/passwordless/webauthn-passkeys[integrating WebAuthn into your application].
